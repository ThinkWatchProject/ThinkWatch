# 授权与商业条款

ThinkWatch 采用 [Business Source License 1.1](LICENSE) 进行源码可见分发。

## 摘要

| 项目 | 规则 |
|------|------|
| 非生产用途 | 可免费使用 |
| 免费生产用途 | 每个 UTC 自然月内，同时不超过 `10,000,000` Billable Tokens 且不超过 `10,000` MCP Tool Calls |
| 商业授权触发条件 | 任一指标超出阈值后，必须购买商业授权 |
| 后续开源切换 | 每个已发布版本会在其 Change Date 或首次公开发布满四周年时自动切换为 `GPL-2.0-or-later`，以较早者为准 |

## 按使用量计费

商业授权按月度使用量计费，而不是按席位计费。

### Billable Tokens

| 计入 Billable Tokens | 不计入 Billable Tokens |
|----------------------|------------------------|
| 生产流量中处理的全部输入 Tokens | 本地开发、CI、测试、预发、演示流量 |
| 生产流量中生成的全部输出 Tokens | 不服务真实生产终端请求的内部评估流量 |
| 同一法律实体名下所有工作区、团队、环境和客户场景的合并总量 | — |

### MCP Tool Calls

| 计入 MCP Tool Calls | 不计入 MCP Tool Calls |
|---------------------|-----------------------|
| 通过 Licensed Work 路由的每一次生产 MCP 工具调用 | 内部健康检查 |
| 无论调用来源是人工用户、Agent、自动化任务还是后台工作流，均计入 | `tools/list` 等工具发现、目录刷新类操作 |
| 同一法律实体名下所有工作区、团队、环境和客户场景的合并总量 | 本地开发、CI、测试、预发、演示流量 |

### 生产使用量

| 术语 | 定义 |
|------|------|
| 生产使用量 | 某个 UTC 自然月内累计的 Billable Tokens 和 MCP Tool Calls |

### 梯度方案

| 档位 | 每月 Billable Tokens | 每月 MCP Tool Calls | 标准价格 |
|------|----------------------|---------------------|----------|
| `Starter` | `0` 到 `10,000,000` | `0` 到 `10,000` | 包含在 [LICENSE](LICENSE) 的 Additional Use Grant 中 |
| `Growth` | `10,000,001` 到 `100,000,000` | `10,001` 到 `100,000` | `USD 499 / 月` |
| `Scale` | `100,000,001` 到 `1,000,000,000` | `100,001` 到 `1,000,000` | `USD 1,999 / 月` |
| `Enterprise` | `1,000,000,001` 到 `10,000,000,000` | `1,000,001` 到 `10,000,000` | `USD 6,999 / 月` |
| `Custom` | 超过 `10,000,000,000` | 超过 `10,000,000` | 定制商业协议 |

| 场景 | 适用档位 |
|------|----------|
| 某月实际适用的商业档位，以 Billable Tokens 和 MCP Tool Calls 两项指标中命中的更高档位为准 | 两项指标中命中的更高档位 |
| `8,000,000` Billable Tokens，且有 `25,000` 次 MCP Tool Calls | `Growth` |
| `220,000,000` Billable Tokens，且有 `80,000` 次 MCP Tool Calls | `Scale` |

- 即使采用月度计量，嵌入式、OEM、托管服务等场景仍可能需要单独的商业协议。
